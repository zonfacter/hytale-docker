# Screenshots

This directory contains screenshots for documentation.

## Required Screenshots

Please add the following screenshots:

1. **setup-wizard.png** - The setup wizard page showing downloader status
2. **setup-oauth.png** - The setup process with OAuth link in the log
3. **dashboard-main.png** - The main dashboard view

## How to Create Screenshots

1. Start the Docker container
2. Open the Dashboard at `http://localhost:8088/setup`
3. Take screenshots at each step of the setup process
4. Save them with the filenames listed above

## Image Guidelines

- Format: PNG
- Width: 800-1200px
- Quality: High (no compression artifacts)
